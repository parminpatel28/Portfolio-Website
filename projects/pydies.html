Pydies

Python study guide for future developers

WHY

I built this project in grade 11 to test my learning of python. More importantly I have seen many individuals struggle with python so I thought I could help them through this project.

HOW

The program was built through the use of python and by applying the basic fundamentals of python. This program has a lot of user interaction to ensure that the user achieves the correct information.

What can you learn

There are 6 chapters to this program which are then broken down into their own section. If you are trying to learn python and feel stuck try out pydies many users have felt more confident with python after using this program. the chapters are listed below feel free to check it out!

  • Basic Python and Console Interaction
  • Conditionals
  • Looping
  • Functions
  • Strings
  • Creating and Altering Data Structures
