Sorting Visualizer

A simple implementation of sorting algorithms

WHY

I built this application because I was fascinated by sorting algorithms, and I wanted to visualize them in action.

HOW

Every time the program starts a new set of array is generated in a random order. The goal is to sort the array in ascending order using one the the sorting algorithms. There are 3 algorithms that will be randomly choosen by the compiler which will be utilized for sorting the array.

Algorithms

1. Bubble Sort

In bubble sort we start at the beginning of the array, as we interate through the array we swap elements that are beside each other in descending order. Then we go to the next pair and continue the same process coninuously making sweeps of the array until it is sorted. in doing so, the smaller items slowly "bubble" up to the beginning of the list. Bubble sort is not the ideal algorithm to use due to the number of comparisions.

2. Insertion Sort

As we begin sorting the array we want to keep the left side of the element we are comparing always sorted. When we interate through the array we check if the left side of the element and if it is lower we would swap it until the entire left side is in ascending order. After we move to the next index and continue the same process.

3. Selection Sort

Selection sort is a simple but inefficient. Using linear scan we would find the smallest element and move it to the front. Then, find the second smallest element and move it, again doing a linear scan. This process is repeated until all elements are in place.
