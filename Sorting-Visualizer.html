Sorting Visualizer

A simple implementation of sorting algorithms

WHY

I built this application because I was fascinated by sorting algorithms, and I wanted to visualize them in action.

HOW

Every time the program starts a new set of an array is generated in random order. The goal is to sort the array in ascending order using one of the sorting algorithms. 3 algorithms will be randomly chosen by the compiler which will be utilized for sorting the array.

Challenges

The major challenge one might experience when making this program is the implementation of the algorithms. However, this was not a challenge for me. The main challenge I've come across was trying to add the GUI. To implement the GUI I first had to learn Java Swing which was difficult due to the complexity of the language. I had to spend a month trying to learn Swing from online resources such as stack-overflow, youtube and w3 schools. I spent ~2 hours almost every day to learn Swing. It has come to my attention that any type of programming you do it will only be efficient if you are willing to do it everyday a for couple of hours.
